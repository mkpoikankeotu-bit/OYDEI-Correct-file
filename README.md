# OYDEI Website

This is the updated Otu Youth Development and Empowerment Initiative (OYDEI) website package.

## Updated content
- Accurate Board of Trustees and Technical Partners
- Previous Events section and gallery content
- Real embedded images extracted from the uploaded OYDEI profile documents
- Admin Dashboard and backend routes retained
- Vercel-ready configuration included

## Deployment on Vercel
1. Upload the contents of this folder to GitHub.
2. Import the repository into Vercel.
3. Use these settings if Vercel asks:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`
4. Add environment variables if you want to change defaults:
   - `ADMIN_PASSWORD` (default is `oydei2026`)
   - Firebase variables if you use your own Firebase project.

## Admin Dashboard
Open the website and click the Admin link. Default admin password: `oydei2026`.
Change this in Vercel environment variables before public launch.
